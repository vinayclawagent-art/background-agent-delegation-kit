# Delegation Promotion Decision Card

> Template-ready only. This card does not claim the Background Agent Delegation Kit has been validated; it turns the next real delegation trial into an explicit promote / pilot-only / iterate / hold decision.

## Required inputs before choosing

- [ ] Filled [[VinClawLabs Delegation Card Trial Packet]] from one real VinClawLabs or Mission Control delegation task.
- [ ] Completed [[Post-Trial Debrief Template]] with actual outputs, review notes, and blockers.
- [ ] Scored [[Delegation Evidence Quality Rubric]] with proof links attached.
- [ ] Prototype export, final artifact link, and reviewer decision are available.

## Decision

Choose exactly one after evidence is attached:

- [ ] **Promote** — repeatable, bounded, produces useful artifact, review gate catches risk.
- [ ] **Pilot-only** — useful in narrow cases; keep as package-local workflow and run 1-2 more trials.
- [ ] **Iterate** — promising, but scope/output/review gate needs redesign before another trial.
- [ ] **Hold** — evidence does not justify continued investment now.
